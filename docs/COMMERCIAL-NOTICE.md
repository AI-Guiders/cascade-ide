# Коммерческие предложения (CascadeIDE)

Исходный код и собранные артефакты этого репозитория распространяются на условиях **MIT** — см. [`LICENSE`](../LICENSE) в корне проекта.

Параллельно команда может предлагать **коммерческие** услуги (поддержка, приоритетные доработки, внедрение, обучение, корпоративные сценарии). Это не меняет лицензию на код в репозитории и не отменяет права по MIT для получателей сборки.
