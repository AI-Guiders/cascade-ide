# Analyzer Rollout: DAL/CCU Boundaries (v1)

Связь: ADR 0097, ADR 0102.

## Новые правила

- `CASCOPE020`: запрет прямой добычи внешних данных в `Cockpit/ComputingUnits/*`.
- `CASCOPE021`: запрет UI-зависимостей в `Cockpit/ComputingUnits/*`.

## Этапы rollout

1. **Warning phase (текущий этап)**
   - Правила включены как `Warning`.
   - Разрешается точечная очистка нарушений вертикальными срезами.

2. **Baseline cleanup**
   - В каждом рефакторинг-батче убираются срабатывания в затронутой области.
   - Добавление новых нарушений в изменяемых файлах не допускается.

3. **Error phase**
   - После стабилизации и очистки основных зон перевести `CASCOPE020/021` в `Error`.
   - Зафиксировать переход в ADR/архитектурной политике и CI.
