# Палитра команд

- **Открыть/закрыть:** по умолчанию **Ctrl+Q** (можно переопределить в `Hotkeys/hotkeys.toml` в каталоге приложения и в `%LocalAppData%\CascadeIDE\hotkeys.toml`; пользовательский файл имеет приоритет).
- **Меню:** Вид → Палитра команд.
- **Закрыть:** Esc, клик по затемнению вокруг панели, повторное сочетание открытия.
- После закрытия фокус возвращается к элементу, который был в фокусе до открытия палитры.

Команды отладки в списке полноценны только в UI-режиме семейства **Debug**; в остальных режимах они отображаются серым и не выполняются по Enter.
