# Фрагменты ADR (`snippets/`)

Переиспользуемые куски Markdown: **один источник правды**, подключаются в нумерованных ADR через `{{ INCLUDE: snippets/… }}` при [сборке книги](../build/README.md).

Подкаталог [`ui/`](ui/README.md) — канон текста для [0076](../0076-ui-ux-principles-hub.md) (центр UI/UX-принципов).

Подкаталог [`tech/`](tech/README.md) — канон текста для [0077](../0077-tech-principles-hub.md) (центр TECH-принципов).

**GitHub / IDE:** без сборки директивы не раскрываются — для чтения в репозитории используй прямые ссылки на файлы здесь или краткое резюме в [README ADR](../README.md).

Имена с префикса **`_`** в `INCLUDE_GLOB` не подхватываются (черновики/служебное).
