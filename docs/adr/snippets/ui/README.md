# `snippets/ui/` — канон для ADR 0076 (UI/UX)

Файлы здесь — **исходники** связного текста «центра» UX. В репозитории **ADR 0076** подключает их через `{{ INCLUDE: … }}`; при правке правь **сниппет**, а не копию в другом месте.

**Ссылки** внутри сниппетов даны **как после сборки** из каталога `docs/adr/build/` (`../0021-…`, `../../design/…`), чтобы HTML/PDF из `build-adr.csx` разрешались корректно. Сырой просмотр только сниппета на GitHub может казаться «кривым» — нормальный путь чтения: **собранный HTML** или файл `0076-*.md` в `docs/adr` (там подставляется развёрнутый текст при сборке).
