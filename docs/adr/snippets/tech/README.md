# `snippets/tech/` — канон для ADR 0077 (TECH)

Файлы здесь — **исходники** связного текста «центра» техники: границы, контракты, агент, отладка, инфраструктура репозитория. В репозитории **ADR 0077** подключает их через `{{ INCLUDE: … }}`; при правке правь **сниппет**, а не копию в другом месте.

**Ссылки** внутри сниппетов даны **как после сборки** из каталога `docs/adr/build/` (`../0008-…`, `../../MCP-PROTOCOL.md`), чтобы HTML/PDF из `build-adr.csx` разрешались корректно. Сырой просмотр только сниппета на GitHub может казаться «кривым» — нормальный путь чтения: **собранный HTML** или файл `0077-*.md` в `docs/adr` (там подставляется развёрнутый текст при сборке).
